# 408_app
一个由作者为助力自身考研写的一个简易版本地数据保存通过邮件或者本地化导出数据文件互传的一个软件，核心除了要使用邮箱互传需要修改注释：EmailJS 初始化配置的相关,即添加上自己的EmailJS配置，使用ai很快的，之后让trae或者其他的agent打包生成对应安装包会更好，由于这里需要使用个人emailjs额度，所有安装包就不直接提供，仅仅提供源码，其次由于本地部署所有不需要后端进行信息互通，所以就没使用后端
